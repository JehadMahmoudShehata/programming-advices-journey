# 🧮 Math Quiz Game

A console-based Math Quiz game built in C++. This project challenges players with arithmetic questions generated dynamically based on chosen difficulty levels and operation types.

## 🚀 Features
* **Custom Quiz Length:** Choose how many questions you want to answer (up to 10).
* **Multiple Difficulty Levels:** Supports Easy, Mid, Hard, and Mix levels.
* **Diverse Operations:** Supports Addition, Subtraction, Multiplication, Division, and Mixed operations.
* **Dynamic Feedback:** System screen color changes based on your answer (Green for Correct, Red for Wrong with a beep sound).
* **Performance Summary:** A complete statistics board displaying final results (Pass/Fail), right/wrong counts, and configurations.

## 🛠️ Concepts Used
* **Language:** C++
* **User-Defined Types:** * `Enums` for operation types and difficulty levels.
  * `Structs` to bundle question details and overall quiz statistics cleanly.
* **Logic & Control:** Functions, arrays of structs, custom random number generation, and switch cases.
