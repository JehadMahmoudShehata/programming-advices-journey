# 🎮 stone-Paper-Scissors Game

A console-based Rock-Paper-Scissors game built in C++. This project demonstrates clean code practices, modular programming, and structural logic separating game configurations from execution.

## 🚀 Features
- **Custom Rounds:** Player can choose to play from 1 to 10 rounds.
- **Random AI:** Computer choices are generated randomly.
- **Visual Feedback:** Screen color changes dynamically based on the round winner (Green for Player, Red for Computer, Yellow for Draw).
- **Detailed Statistics:** Complete game summary shown at the Game Over screen.


## 🛠️ Concepts & Technologies Used

* **Language:** C++
* **User-Defined Types:** * `Enums` (for choices: Stone, Paper, Scissor, and winners).
  * `Structs` (to bundle and manage round data and final game results cleanly).
* **Logic & Control:** Functions, loops, and input validation.
